# Public product/catalogue pages for the Suplymate product scraper.
#
#   npm run products:scrape -- --file=scripts/import/examples/product-catalogs.txt --limit=150
#   npm run products:import
#
# FORMAT: one entry per line. Lines starting with "#" are ignored.
#   URL
#   URL | Category
#   URL | Category | supplier-id
# Category must be one of: Steel & Metals, Cables & Electrical, Tubes & Pipes,
# Packaging, Construction, Industrial Parts. When omitted it is inferred from the
# product name. When supplier-id is omitted the product is linked by domain (a
# PENDING supplier is created if none exists). All output is PENDING until an
# admin publishes it at /admin/products.
#
# COMPLIANCE: list ONLY public catalogue pages you may index. The scraper checks
# robots.txt + Crawl-delay before every request, rate-limits, refuses
# social/private/auth/checkout pages, and never bypasses anti-bot protection.
