Suplymate website files

Open index.html in your browser to start.
Pages included:
- index.html: landing page
- search.html: supplier search results
- supplier.html: supplier profile
- pricing.html: pricing page
- login.html: demo login (sessionStorage)
- signup.html: create account (localStorage demo)
- forgot-password.html: password reset request (demo)
- dashboard.html: buyer dashboard (auth required)

Assets are in /assets, including favicon and logo icon.
